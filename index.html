📅 Connect Outlook Calendar
Pulls past meetings with Merve and Zeynep from your Outlook calendar via Microsoft Graph.
1. Go to portal.azure.com → Azure Active Directory → App registrations → New registration
2. Name it anything · Accounts: "Accounts in any org directory + personal Microsoft accounts"
3. Redirect URI → Web → http://localhost:3020
4. After registering: API permissions → Add → Microsoft Graph → Delegated → Calendars.Read
5. Paste the Application (client) ID here

Connect once - then push any task to Asana with one click.
Your token is stored only in this browser (localStorage). Never shared.
Open your Asana project → copy the number from the URL: app.asana.com/0/PROJECT_GID/...
